# Escape This Class

A VR escape room game developed in Unity for the Meta Quest 2, where players must solve puzzles to escape from a classroom.

## Overview

**Escape This Class** is an immersive virtual reality escape room experience set in a classroom environment. Players must explore, interact with objects, and solve various puzzles to find their way out. The game features intuitive VR controls and engaging puzzle mechanics designed specifically for the Meta Quest 2.

## Features

- **Full VR Immersion** - Built for Meta Quest 2 with hand tracking and controller support
- **Interactive Environment** - Pick up, examine, and manipulate objects in the classroom
- **Diverse Puzzles** - Multiple puzzle types including:
  - Programming logic puzzles
  - Pattern recognition challenges
  - Object manipulation tasks
  - Hidden clue discovery
- **Physics-Based Interactions** - Realistic object handling using Unity's physics system
- **Atmospheric Design** - Detailed classroom environment with ambient lighting

## Technologies Used

- **Engine:** Unity (Universal Render Pipeline)
- **Platform:** Meta Quest 2 (Oculus)
- **VR Framework:** XR Interaction Toolkit
- **Physics:** Oculus Hands Physics
- **Language:** C#

## How to Play

### Controls
- **Grip Button:** Grab and hold objects
- **Trigger Button:** Interact with UI elements and buttons
- **Joystick:** Teleport around the room
- **Hand Tracking:** Natural hand gestures for interaction (optional)

### Objective
Explore the classroom, find clues, solve puzzles, and unlock the door to escape!

##  Getting Started

### Prerequisites
- Unity 2021.3 or later
- Meta Quest 2 headset
- Oculus Developer account (for building to device)

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/Escape-This-Class.git
   ```
2. Open the project in Unity Hub
3. Open the main scene located in `Assets/Scenes/`
4. Connect your Meta Quest 2 via Oculus Link or build directly to the device

### Building for Quest 2
1. Go to File > Build Settings
2. Switch platform to Android
3. Connect your Quest 2 and enable Developer Mode
4. Click "Build and Run"
